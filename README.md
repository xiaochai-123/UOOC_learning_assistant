# UOOC 2025.11学习助手

## 脚本简介

本脚本专为 UOOC 优课联盟课程平台打造，可实现全自动化学习体验。支持连续播放、自动倍速、自动静音、小节连播、自动跨章节跳转、全自动答题(视频中而非测验)，并针对答题弹窗做了智能关闭优化，兼容大多数 UOOC 课程结构。智能递归播放，极大提升学习效率。
## 功能清单

- 视频自动播放，无需手动点击
- 自动切换到二倍速播放，加快学习效率
- 视频自动静音，学习不打扰别人
- 自动跳转至下一个未完成视频，支持自动展开章节
- 自动答题，智能识别题目并快速提交，仅答一次防死循环
- 答题弹窗自动关闭，避免卡死或影响连播流程
- 支持多轮自动检测，兼容延迟加载章节和视频
- 刷完本章节自动进入下一个章节，直至全部课程完成

## 安装方法

1. 请确保浏览器已安装 [Tampermonkey](https://www.tampermonkey.net/) 或其他用户脚本扩展。
2. 复制本脚本全部内容，或者点击 油叉/Greasy Fork 安装按钮。
3. 保存后，打开 UOOC 优课联盟课程网站学习页面，脚本会自动运行。

## 使用说明

- 脚本启动后自动检测未完成的视频任务并开始播放，无需手动干预，自动完成全部课程
- 自动倍速、静音、答题、弹窗关闭等功能全部自动触发
- 如遇页面结构更新或特殊变动，欢迎在油叉/Greasy Fork 或 GitHub 留言反馈，我将及时优化脚本兼容性

## 常见问题 & 注意事项

- 仅支持 UOOC 优课联盟课程平台，其他网站无效
- 脚本自动跳过章节测试题（如需要手动完成请注意）
- 若脚本出现无法自动连播、答题、弹窗无法消除等情况，请刷新页面或反馈具体问题
- 本脚本只可答视频中的弹窗题目，无法自动回答测验题目，测验请手动完成
